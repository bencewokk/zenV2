# Zen Web Capture

Load this directory as an unpacked Chromium extension:

1. Open `chrome://extensions` (or the equivalent extensions page).
2. Enable **Developer mode**.
3. Choose **Load unpacked** and select this `browser-extension` directory.
4. Capture a page or selection, then open **Zen → Sources → Import web capture**.

Captures are plain `.zenclip.json` files. They preserve the original URL, title,
author metadata, capture time, selection, and a clean-text page extract.

# Zen 3.2.0

## Spark Intro

- A new first-run intro plays before the connection walkthrough: ignite, pick your app look, then watch every tab open.
- Replay it anytime from Settings → Connections → **Replay intro**.

## Livelier shell

- The header is now an animated card nav, hosting search, notes, AI, and Deep Work controls.
- Settings gets an animated sidebar for section navigation.
- The Sources panel opens on a tiled hub for picking a provider to connect, instead of a blank state.

## Fixes

- The auto-updater now correctly detects new releases — the update-check proxy was misreading GitHub's asset URLs and reporting "no release" even when one existed.

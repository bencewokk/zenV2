GitHub now connects at the account level instead of asking for repository names.

- Zen discovers every repository visible to the connected token: owned, collaborated, and permitted organization repositories.
- Removed the manual `owner/repository` list from Connections.
- The token remains the permission boundary: choosing **All repositories** at GitHub gives Zen account-wide access, while a restricted token limits Zen automatically.
- Every repository, its recent issues and pull requests, and a bounded set of readable source/documentation files are indexed without exhausting GitHub's API quota.
- Background source refresh now runs hourly, while **Refresh all** remains available for immediate updates.

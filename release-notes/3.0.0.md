A milestone release focused on making first impressions real, not decorative.

- **One-click "Sign in with Google & sync everything"** on the very first
  onboarding screen — signs in, turns on cloud sync, and runs an initial sync
  in a single click for anyone who just wants to connect and go.
- Fixed a bug where the walkthrough's Google **Connect** button stayed
  disabled on desktop installs with no client ID typed in yet, even though
  the desktop app always has a usable bundled connection.
- The **"Everything you can do" feature gallery** no longer shows static
  screenshots. All six cards are now real, interactive demos: an editable
  math field, a draggable geometry board, an inline preview of the real
  bundled sample PDF, a live AI rewrite (or a graceful canned example if AI
  isn't connected yet), a real link between two sample notes you can click
  into, and today's real daily brief when one exists.
- Added a second seeded sample note ("Sample: Functions") so the wiki-link
  demo has something real to link to.

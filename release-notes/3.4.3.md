# Zen 3.4.3 — Aurora

## First Run Path that keeps up

- Checklist goals now tick when you actually complete their matching actions: using search, arranging a Deep Work window, opening Study, setting a daily goal, opening Settings, and re-quizzing mistakes.
- Guided lessons now have one clear start goal instead of two duplicate checklist entries.
- Seeded sample notes no longer complete organising or authoring goals before you have done the work yourself.
- Tutorial progress writes are merged safely, so an achievement recorded outside the dashboard cannot be overwritten by stale dashboard state.

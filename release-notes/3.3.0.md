# Zen 3.3.0

## Spark setup

- Spark now owns first-run setup: choose a look, connect Google, enable or skip sync, review connected sources, and seed a private profile in one flow.
- The old separate onboarding walkthrough is gone, so first launch no longer stacks two guided experiences.
- Settings now replays the single Spark setup flow instead of splitting intro and walkthrough replay buttons.

## Cleaner first-run state

- Release notes now key off Spark's first-run flag, so new installs avoid stacked modals without depending on removed onboarding state.
- The capabilities inventory now reflects Spark setup as the first-run path.

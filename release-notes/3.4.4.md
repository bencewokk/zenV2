# Zen 3.4.4 — Aurora

## First Run Path that reflects what you did

- The seeded sample Deep Work session and its sources no longer pre-complete "create or open a session", "add a source", or "gather a second source" — only sessions and sources you make yourself count.
- Naming a Deep Work session now completes its goal (previously only setting an intent did).
- Trying search, exporting a backup, copying diagnostics, and opening Plan & usage now tick their matching goals.
- Starting a guided lesson now also completes "Try a lesson/class".
- The sample PDF step is optional, so Collect Material no longer stalls when you skip it — and its action now points you to create your first note instead of jumping to search.

## A dashboard that fits the window

- On narrow or small windows the notes sidebar now steps aside and the tile grid reflows, instead of getting crushed into unreadable columns.

## Build identity

- The status bar now shows the short Git commit beside the app version, making it easy to identify the exact build during testing and support.

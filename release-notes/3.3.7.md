# Zen 3.3.7

## Guided walkthroughs

- Added interactive, spotlight-guided walkthroughs to the First Run Path. Click **Start walkthrough** on a step and Zen coaches you through it hands-on — dimming the screen, highlighting the real button, and waiting for you to actually do each task.
- Action steps advance only when you complete them, then show a short "✓ Done" confirmation before moving on. The spotlight follows focus onto whatever you open (search, the PDF picker, the source library).
- Shipped three walkthroughs: **Collect Material** (create and name a note, add a PDF, search), **Start Deep Work** (open a session, add sources, arrange the canvas), and **Study & Quiz** (open the Study panel, run a focus block, quiz yourself).
- Replay any walkthrough from the First Run Path, or the core loop from Settings → Appearance.

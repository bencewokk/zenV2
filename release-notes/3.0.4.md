Google Drive now connects at the account level instead of asking for individual folder links.

- Removed the Drive folder URL/ID field and the confusing homepage-link 404 failure.
- Zen now indexes every non-trashed file visible to the signed-in Google account, including nested content and shared-drive items.
- Supported Docs, Slides, Sheets, and text files are extracted in bounded parallel batches; other formats retain searchable metadata and original links.
- Drive access remains read-only and continues to use the Google identity already connected to Zen.

# Zen 3.4.0

## Zen Assistant

- New assistant backend: background Gmail and Calendar routines now run server-side on your schedule, wake up for due runs, and can reach you through Web Push notifications.
- The PWA's Google sign-in can hand Zen encrypted offline credentials, so routines keep working when the app is closed. Refresh tokens and push subscriptions are encrypted with the connection vault key before they are stored.
- Assistant captures, tasks, routines, and receipts sync into the app. Review, import, and tick them off from Settings → Data.
- The in-app AI gained tools for working with assistant tasks and captures.

## Learn Zen by doing

- The dashboard tutorial is now hands-on. **Do the core loop** runs a ~2-minute interactive tour where you actually create and name a real note, write in it, jump to it with search, and put it on a Deep Work canvas — each step advances when you do it, not when you click Next.
- Every unfinished goal on the First Run Path is now a live entry point: click it and Zen starts the guided walkthrough for that phase. The main call-to-action always leads into the next walkthrough.
- If a walkthrough highlight sits outside the view, the tour scrolls it into view and, if you scroll away again, pins a cue at the screen edge pointing toward it — click the cue to jump back.

# Zen 3.0.6

## Google-first onboarding

- Rebuilt first-run setup around one Google identity connection.
- Removed user-supplied Google OAuth client fields; released builds always use Zen's built-in OAuth client.
- Added an explicit cloud-sync choice between cross-device sync and local-only storage.
- Requires each optional source to be connected or deliberately marked "Not now" before setup continues.
- Requires an explicit AI connection or skip decision and removes the global silent skip path.

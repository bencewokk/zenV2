# Zen 3.0.9

## Subscription-backed AI enforcement

- Reads Free, Basic, and Plus access from the website-managed account tied to your Google identity.
- Free users are stopped before any provider or quota check.
- Basic runs DeepSeek V4 Flash with a $5 monthly budget.
- Plus runs DeepSeek V4 Pro (and can switch to Flash) with a $25 monthly budget.
- Routes model calls through Zen's authenticated backend so limits cannot be bypassed by modifying the desktop client.
- Atomically reserves monthly dollar budget, settles actual token cost, and rolls back provider failures.
- Adds a Plan & Usage screen with the current tier, billing month, caps, and consumption.
- Expands Plan & Usage into a dashboard with spend, remaining budget, request count, assigned model, renewal date, progress, and per-model breakdown.

## Private updates

- Routes signed update metadata and installer downloads through Zen's backend, keeping the private-repository token off user devices.
- Publishes release builds immediately so the updater never points at a draft release.

## New first-run setup

- Replaces the old feature-demo tour with a seven-step Google-first account setup.
- Detects your website-managed Free, Basic, or Plus plan without asking for provider keys.
- Explains the plan's DeepSeek V4 model and monthly dollar budget before AI is used.
- Requires explicit sync, source, and profile choices, then shows a final setup review.
- Adds a complete local-data reset for testing the first-run experience without deleting the cloud account or subscription.

AI replies now start sooner and stream more smoothly: context retrieval has a strict latency budget, server authentication and billing do less serial work, and streamed updates render in batches. Stopping or switching conversations cannot redirect late replies, and already-dispatched actions are never repeated automatically when completion is uncertain.

Sync now resolves concurrent edits transactionally across clients and the API, protects active note drafts from remote overwrites, and applies stable last-write-wins ordering. Assistant writes, scheduled routines, and undo operations use durable claims so duplicate requests cannot execute the same side effect twice.

Backups and settings transfers now exclude credentials, preserve device secrets only when the provider identity still matches, validate imported data, and describe omitted PDF/source data accurately.

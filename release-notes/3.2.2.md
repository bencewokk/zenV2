# Zen 3.2.2

## Fixes

- **PDFs open again.** Opening a PDF could show "This content is blocked" instead of the document. The viewer now loads correctly.
- **No more black screen when studying.** Starting a lesson or study session could turn the whole app black on some machines. Zen now stays put, and if a visual effect can't run it's quietly skipped instead of taking the app down.

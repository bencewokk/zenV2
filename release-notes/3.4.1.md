# Zen 3.4.1 — Aurora

## Zen on your phone

- New dashboard tile with a QR code that connects your phone: scan it and the Zen Assistant offers a one-tap install (Android) or Add-to-Home-Screen steps (iPhone), then signs in with your Google account to link back to this Zen.
- A Copy link button shares the same install link over any channel.

## Release codenames

- Releases now carry a codename — this one is **Aurora**. Codenames show in What's new and the release notes; versions stay plain semver under the hood.

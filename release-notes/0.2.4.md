- Release notes now live in /release-notes as one markdown file per version, surfaced in a dashboard "What's new" card that auto-opens once after each update.
- Versioning reworked into a single source of truth: the version is derived from the latest git tag and flows automatically to the app, the Tauri bundle, and the auto-updater — nothing is hand-edited.
- A post-commit git hook now turns every commit into its own release: it writes a note from the commit message and tags the version.

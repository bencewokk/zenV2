# Zen 3.3.1

## Dashboard first-run path

- Added a compact dashboard tutorial that guides new users through the real Zen loop: setup, material, Deep Work, study, quiz, connected context, and trust controls.
- Tutorial ticks complete automatically when Zen can detect progress, with manual ticks for actions that are intentionally lightweight or outside a single detectable event.
- The next-action button opens the right surface, such as Spark setup, search, Deep Work, Sources, or Settings.

## Spark setup

- Reworked Spark into a focused onboarding path for look, account, sync, source, and profile setup.
- Added Canvas, Zotero, and GitHub credential fields directly inside Spark so users can connect integrations without leaving first run.
- Removed the old Spark app-surface tour now that the dashboard tutorial carries the product walkthrough.

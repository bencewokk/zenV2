# Zen 3.1.0

## Academic reasoning upgrades

- Adds Pyodide-backed SymPy support for stronger math simplification, equivalence, and derivation checks.
- Reworks math notes around step-by-step derivations instead of hidden expected-answer targets.
- Keeps old math blocks compatible while hiding answer keys from the note experience.
- Adds a full app capability inventory for planning the next wave of academic features.

## Faster navigation

- Adds a global command palette on Ctrl/Cmd+K.
- Searches notes, PDFs, connected sources, and Deep Work sessions.
- Includes quick actions for new notes, Deep Work, Calendar, Mail, Sources, and Settings.

## First-run and daily UX polish

- Adds undoable note deletion.
- Prevents first-run onboarding from stacking with release notes.
- Cleans up onboarding plan names and removes stale API-key copy.
- Adds a one-click "Skip all for now" path for optional sources.
- Makes status badges open Settings.
- Cascades new Deep Work windows so they do not spawn on top of each other.

## Honest AI availability

- Adds an app-wide AI access check for signed-out and free-tier states.
- Disables chat, tutor, quiz, and planning actions when AI cannot work.
- Shows clear recovery copy with an Open Settings path.
- Lets non-AI study tools keep working when AI is unavailable.
- Falls back silently for decorative daily quotes instead of showing startup error toasts.
